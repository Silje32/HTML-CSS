How to link a stylesheet

To link an external stylesheet you can use the link tag inside the head element. The rel attribute explains the relation the link has to our document. The value in this case will always be stylesheet, since that is what weÂ´re creating a link to.

The href attribute is the link to our stylesheet.

link rel ="stylesheet" href ="styles.css"

Margin

Margin is actually the space around an element.

The browser targets the body and sets a margin of 8px as the default. No element touches the edges of the browser.

To make this default margin to go away set the margin to 0 in the body.

body { margin: 0; }
